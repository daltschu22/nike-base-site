About Nike Missile Base Map

Project Overview

The Nike Missile Base Map is an interactive web application that displays the locations of Nike missile sites across the United States. This project aims to preserve the historical significance of these Cold War-era defensive installations by making their locations and information easily accessible.

Nike Missile System History

The Nike missile system was a line of American anti-aircraft missile systems designed and deployed from the 1950s to the 1970s. Named after the Greek goddess of victory, the Nike system was developed to defend against Soviet bomber attacks during the Cold War.

The system consisted of two primary components:

  • Launch Area: Where the missiles were stored and launched
  • Control Area: Where radar and computer systems for tracking targets and guiding missiles were located

The Nike system went through several iterations, including:

  • Nike Ajax: The first operational anti-aircraft missile system (1954)
  • Nike Hercules: An improved version with nuclear capability (1958)
  • Nike Zeus: An anti-ballistic missile system (never fully deployed)

By the 1970s, most Nike missile sites were decommissioned as they became obsolete with the development of intercontinental ballistic missiles (ICBMs) and changing military strategies.

Data Sources

The data for this application is sourced from:

  • Wikipedia: List of Nike missile sites

The application uses a web scraper to extract site information, including coordinates, names, and descriptions from the Wikipedia page.

How to Use the Map

  1. Import Data: Click the "Import Data" button to fetch Nike missile site data from Wikipedia.
  2. Explore the Map: Navigate the map to see all Nike missile sites marked with pins.
  3. View Site Details: Click on any marker to view detailed information about that site.
  4. Filter Sites: Use the filters panel to narrow down sites by state or site type.
  5. Learn More: Click the "More Info" button in the site details panel to visit the Wikipedia page.

Technical Information

This application is built using:

  • Backend: Python with Flask framework
  • Database: SQLite with SQLAlchemy ORM
  • Frontend: HTML, CSS, JavaScript, Bootstrap 5
  • Mapping: Google Maps JavaScript API
  • Data Processing: BeautifulSoup for web scraping
